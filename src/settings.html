ADM

Automatic Deployment Model

ADM 是一个 llama.cpp 图形化管理工具，旨在将 llama.cpp 复杂的 CLI 启动指令通过界面化配置，让用户能够便捷地在本地部署和运行大语言模型。

基于 Tauri 2.x 构建，采用原生 HTML/CSS/JS 前端 + Rust 后端架构，轻量高效。
